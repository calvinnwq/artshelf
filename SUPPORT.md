# Support

Use [GitHub issues](https://github.com/calvinnwq/artshelf/issues) for Artshelf
support:

- Bug report: something implemented behaves incorrectly.
- Feature request: a small addition that fits Artshelf's v1 safety model.
- Question: usage, setup, or cleanup-plan behavior.

Artshelf is an MVP, so support is best-effort. Please include the command you ran,
your Node version, and whether you used a repo-local, user-global, or explicit
ledger.
