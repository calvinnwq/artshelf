# Support

Use GitHub issues once the public remote exists:

- Bug report: something implemented behaves incorrectly.
- Feature request: a small addition that fits Shelf's v1 safety model.
- Question: usage, setup, or cleanup-plan behavior.

Shelf is an MVP, so support is best-effort. Please include the command you ran,
your Node version, and whether you used a repo-local, user-global, or explicit
ledger.
