Install

Install Artshelf.

Artshelf is being prepared for npm distribution under the unscoped `artshelf` package name. Source install remains the fallback path.

install
$ npm install -g artshelf
$ artshelf --version
$ artshelf doctor

Requirements

Node.js 22+

The package declares `node >=22` and CI runs on Node 24.

Corepack

Use Corepack to activate the pinned pnpm version from `packageManager`.

Git

Clone the public repo and keep source installs easy to update with `git pull`.

Install From npm

npm install -g artshelf
artshelf --version
artshelf doctor
pnpm add -g artshelf
artshelf --version
artshelf doctor
npm uninstall -g artshelf

Install From Source

git clone https://github.com/calvinnwq/artshelf.git
cd artshelf
corepack enable
pnpm install --frozen-lockfile
pnpm run build
npm link
artshelf --version
artshelf doctor

`npm link` connects the local checkout to your global npm bin, so future rebuilds update the `artshelf` command. 

npm unlink -g artshelf

Update A Source Install

cd artshelf
git pull
pnpm install --frozen-lockfile
pnpm run build
npm link
artshelf --version
artshelf doctor

Agent Setup

Agents should prefer the npm install when available. If using a source install, ask the user where to clone it before linking the command. 

docs/agent-usage.md
Verify `artshelf --version` and `artshelf doctor` after either install method.

Preview Docs Locally

pnpm docs:serve

This serves the static docs directory at 127.0.0.1:8080.
