Source install

Install from source.

Shelf 0.1.0 is GitHub/source-first. The supported path is a local clone, build, and `npm link`.

install
$ git clone https://github.com/calvinnwq/shelf.git
$ cd shelf
$ corepack enable
$ pnpm install --frozen-lockfile
$ pnpm run build
$ npm link
$ shelf --version

Requirements

Node.js 22+

The package declares `node >=22` and CI runs on Node 24.

Corepack

Use Corepack to activate the pinned pnpm version from `packageManager`.

Git

Clone the public repo and keep source installs easy to update with `git pull`.

Install Locally

git clone https://github.com/calvinnwq/shelf.git
cd shelf
corepack enable
pnpm install --frozen-lockfile
pnpm run build
npm link
shelf --version

This is the only supported install method for now. `npm link` connects the local checkout to your global npm bin, so future rebuilds update the `shelf` command. 

npm unlink -g shelf

Update A Source Install

cd shelf
git pull
pnpm install --frozen-lockfile
pnpm run build
npm link
shelf --version

Agent Setup

Agents should use the same local source install. If Shelf is not installed, ask the user where to clone it before linking the command.
