Source install

Install from source.

Shelf is not on npm yet. The supported pre-release path is a source checkout with Corepack, pnpm, and an optional local global link.

install
$ git clone https://github.com/calvinnwq/shelf.git
$ cd shelf
$ corepack enable
$ pnpm install --frozen-lockfile
$ pnpm run build
$ node dist/src/cli.js --version

Requirements

Node.js 22+

The package declares `node >=22` and CI runs on Node 24.

Corepack

Use Corepack to activate the pinned pnpm version from `packageManager`.

Git

Clone the public repo and keep source installs easy to update with `git pull`.

Build From Source

git clone https://github.com/calvinnwq/shelf.git
cd shelf
corepack enable
pnpm install --frozen-lockfile
pnpm run build
node dist/src/cli.js --version

The direct Node invocation is the most explicit pre-release smoke test. It proves the compiled CLI works without adding a global command to your machine.

Optional Global Link

npm link
shelf --version

This mirrors the common source-install flow for small Node CLIs. It links the local checkout into your global npm bin, so future local rebuilds update the linked command. 

npm unlink -g shelf

Update A Source Install

cd shelf
git pull
pnpm install --frozen-lockfile
pnpm run build
shelf --version

Expected npm Install

After the first release, the intended package install will be:

npm install -g shelf
shelf --version
Do the source install and live smoke tests first. Publishing comes later.
